# Stellarbeat frontend

Frontend

Built with the https://vuejs.org/ framework. (Migration to vue3 is in progress,
currently on 2.7)

## Usage

### installation

`pnpm install`

Copy .env.dist to .env and fill in the values.

### development mode

`pnpm dev`

### production build

production build: `pnpm build`
